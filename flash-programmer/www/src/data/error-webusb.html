This browser does not support WebUSB, so you won't be able to use it with your Floopy Drive. Please try a
supported browser.

Some alternatives exist as well: You can install some WebUSB capable browsers as a portable (temporary) solution, if that is preferred. Most WebKit-based browsers should support it. The chrlauncher Chromium launcher for Windows makes it a snap to retrieve, launch, and cleanly remove Chromium browsers. Or, for Chrome without any of the Google nonsense, check out Ungoogled Chromium for Linux/Mac/Win.
